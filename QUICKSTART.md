# MarketGenius - Quick Start Guide\n\n## Installation & Setup\n\n### Prerequisites\n- Python 3.8 or higher\n- pip (Python package installer)\n- Windows PowerShell or Command Prompt\n\n### Step 1: Clone/Setup Project\n```bash\ncd d:\\nifty-stock-predictor\n```\n\n### Step 2: Backend Setup\n\n**Create Virtual Environment:**\n```bash\ncd backend\npython -m venv .venv\n.\\.\venv\\Scripts\\Activate.ps1\n```\n\n**Install Dependencies:**\n```bash\npip install -r requirements.txt\n```\n\nIf `requirements.txt` doesn't exist, install manually:\n```bash\npip install flask flask-cors pandas numpy scikit-learn chart-js\n```\n\nOptional ML libraries (for enhanced predictions):\n```bash\npip install xgboost lightgbm tensorflow\n```\n\n### Step 3: Prepare Data\n\nPlace your market data CSV files in:\n```\nd:\\market dataset\\\n```\n\nExpected files:\n- `NIFTY_50.csv`\n- `SENSEX.csv`\n- `NIFTY_50_COMPANIES.csv`\n- `SENSEX_COMPANIES.csv`\n\nCSV Format (required columns):\n```\nDate,Open,High,Low,Close,Volume\n2024-01-01,19500.50,19750.75,19400.25,19600.00,1000000\n```\n\n### Step 4: Start Backend\n\n```bash\npython app.py\n```\n\nYou should see:\n```\nLoading data from CSV files...\nTraining prediction models...\nStarting Flask server...\n * Running on http://0.0.0.0:5000\n```\n\n### Step 5: Open Frontend\n\n**Option A: Using Flask (Recommended)**\n- Open browser and navigate to: `http://127.0.0.1:5000`\n- Flask serves both frontend and API\n\n**Option B: Using Live Server**\n- Right-click `frontend/index.html`\n- Select \"Open with Live Server\"\n- Automatically opens `http://127.0.0.1:5500`\n- Backend API on port 5000 required\n\n## Features Quick Reference\n\n### Dashboard\n- View market overview\n- Check watchlist prices\n- Analyze main chart with indicators\n- Switch between line and candlestick views\n\n### Analysis Tab\n- Deep technical analysis\n- Professional candlestick charts\n- Multiple indicators display\n- Symbol-specific analysis\n\n### Predictions\n- Select a future date\n- Get AI price predictions\n- See change percentage\n- Buy/Hold/Sell signals\n\n### Investment Analysis\n- Returns analysis\n- Volatility metrics\n- Technical setup\n- Investment recommendations\n\n### Market News\n- Real-time news feed\n- Sentiment analysis\n- Trading signals\n- Refresh button for updates\n\n## Troubleshooting\n\n### \"Failed to fetch\" Error\n**Solution:**\n- Ensure backend is running on port 5000\n- Check firewall isn't blocking Python\n- Verify CORS is enabled in Flask\n\n### \"404 Not Found\" for CSV files\n**Solution:**\n- Create `d:\\market dataset` folder\n- Add required CSV files\n- Ensure correct column names\n\n### Charts not loading\n**Solution:**\n- Clear browser cache (Ctrl+Shift+Delete)\n- Check browser console for errors\n- Ensure JavaScript libraries loaded\n\n### Candlestick chart not appearing\n**Solution:**\n- Click \"Candlestick\" button in Dashboard\n- Ensure data is loaded\n- Check browser console for errors\n\n## Performance Tips\n\n1. **Large Datasets**\n   - Limit history to recent 1-2 years\n   - Use higher timeframes (1M, 3M)\n\n2. **Multiple Users**\n   - Run Flask with `debug=False`\n   - Use production WSGI server (Gunicorn)\n\n3. **Memory Management**\n   - Reduce number of loaded symbols\n   - Clear old data periodically\n\n## Advanced Configuration\n\n### Enable Real News API\n\nEdit `backend/app.py` line ~1030:\n```python\n'apiKey': 'YOUR_NEWSAPI_KEY'  # Get from newsapi.org\n```\n\n### Customize Indicators\n\nEdit `frontend/script.js` indicators config:\n```javascript\nconst indicators = {\n  sma: { periods: [20, 50, 200], enabled: false },\n  ema: { periods: [9, 21], enabled: false },\n  rsi: { period: 14, enabled: false },\n  macd: { enabled: false }\n};\n```\n\n### Change API Port\n\nEdit `backend/app.py` last line:\n```python\napp.run(debug=False, host='0.0.0.0', port=5000)  # Change 5000 to desired port\n```\n\nAnd update frontend `script.js` line 4:\n```javascript\nconst DEFAULT_API = 'http://127.0.0.1:5000';  // Update port\n```\n\n## Browser Compatibility\n\n✅ Supported:\n- Chrome 80+\n- Firefox 75+\n- Edge 80+\n- Safari 13+\n\n❌ Not Supported:\n- Internet Explorer\n- Old mobile browsers\n\n## Security Notes\n\n1.
