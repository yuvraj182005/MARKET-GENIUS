│\n└────────────────┬────────────────────────┘\n                 ↓\n        ┌────────────────┐\n        │ Success? Yes   │→ Display Real Data\n        │        │       │→ Continue updates\n        │        │       │→ Show 🟢 Connected\n        │   No   ↓       │\n        └────────────────┘\n                 ↓\n┌─────────────────────────────────────────┐\n│  Error Caught in Try-Catch              │\n│  (Network error, API error, etc)        │\n└────────────────┬────────────────────────┘\n                 ↓\n┌─────────────────────────────────────────┐\n│  Backend Fallback Logic                 │\n│  • If real API fails                    │\n│  • Return mock/cached data              │\n│  • Log error for debugging              │\n└────────────────┬────────────────────────┘\n                 ↓\n┌─────────────────────────────────────────┐\n│  Frontend Receives Response              │\n│  • Even if error, got data to display   │\n│  • Page doesn't crash                   │\n│  • Status shows 🔴 Disconnected         │\n└────────────────┬────────────────────────┘\n                 ↓\n┌─────────────────────────────────────────┐\n│  Log Error to Console (F12)             │\n│  • Helps with debugging                 │\n│  • User doesn't see stack traces        │\n│  • Friendly error message shown         │\n└────────────────┬────────────────────────┘\n                 ↓\n┌─────────────────────────────────────────┐\n│  Automatic Retry                        │\n│  • Retry in 5 seconds for quotes        │\n│  • Retry in 30 seconds for status       │\n│  • Backoff on repeated failures         │\n│  • Auto-recover when connection restored│\n└─────────────────────────────────────────┘\n```\n\n---\n\n## File Relationships\n\n```\nindex.html (Main Page)\n    ↓\n    ├─→ script.js\n    │   └─→ Calls initAngelOne() after page load\n
