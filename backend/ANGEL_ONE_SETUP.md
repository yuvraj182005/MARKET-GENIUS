# Angel One SmartAPI Configuration\n\nThis guide helps you integrate Angel One SmartAPI (official Indian trading API) with MarketGenius.\n\n## What is Angel One SmartAPI?\n\nAngel One SmartAPI is India's official trading API by Angel One (formerly Angel Broking).\n- Real-time market data\n- Live stock prices\n- Trading capabilities\n- Official NSE/BSE data\n- 24/5 support\n\n## Setup Steps\n\n### 1. Create Angel One Account\n1. Visit: https://www.angelone.in/\n2. Sign up for free account\n3. Verify KYC (if trading)\n\n### 2. Get API Credentials\n1. Login to Angel One website\n2. Go to: Profile → API → SmartAPI\n3. Generate API key\n4. Save: Client ID, API Key, Password\n\n### 3. Configure Backend\n\nCreate `.env` file in `backend/` with:\n```env\nNEWSAPI_KEY=your_newsapi_key\nANGEL_ONE_CLIENT_ID=your_client_id\nANGEL_ONE_API_KEY=your_api_key\nANGEL_ONE_PASSWORD=your_password\nANGEL_ONE_TOTP=000000\n```\n\n### 4. Install SmartAPI SDK\n```bash\npip install smartapi-python\npip install pyotp\n```\n\n### 5. Restart Backend\n```bash\npython app.py\n```\n\n## Features\n\n✅ Real-time NSE/BSE quotes\n✅ Live LTP (Last Trading Price)\n✅ OHLC data\n✅ Market depth\n✅ Trade execution (if enabled)\n✅ Portfolio tracking\n✅ Order history\n\n## API Endpoints Added\n\n- `GET /angel-data` - Market data from Angel One\n- `GET /angel-quotes?symbols=TCS,RELIANCE` - Real quotes\n- `GET /angel-portfolio` - Your portfolio\n- `POST /angel-order` - Place orders (optional)\n\n## Testing\n\n### Without API Key\n- App works with mock data\n- All features functional\n- Perfect for development\n\n### With API Key\n- Real-time market data\n- Live trading quotes\n- Portfolio information\n\n## Security\n\n⚠️ **Never share credentials!**\n- Store in `.env` (never commit)\n- Use `.env.example` as template\n- Add `.env` to `.gitignore`\n\n## Support\n\n- Angel One Docs: https://www.angelone.in/api-documentation\n- SmartAPI SDK: https://github.com/angelbroking-github/smartapi-python\n- Support: support@angelone.in\n
